Vitalisera inventering ?

Inventeringsapp för Vitalisera. Skanna QR-koder med kameran eller välj artiklar i listan för att registrera inventering. Tryck + för att skapa en ny artikel manuellt.
Välj artikel i listan

Tryck på rubrikerna för att byta ordning.

Ange ditt namn ?

Ditt namn sparas lokalt i 10 dagar och visas i Sheetet vid varje registrering, så att man ser vem som inventerade.

"Fler fält" visar kommentar, enhet, typ och min-antal.

Filtrera listor ?

Välj vilka platser som ska visas i listorna. Kryssa i "Visa endast artiklar med för lågt saldo" för att bara se artiklar som behöver fyllas på. Tryck "Använd" för att spara.

Sök artiklar ?

Sök artiklar med fritext. Tryck på ett resultat för att öppna artikeln. Tryck "Ny artikel" för att skapa en helt ny.

Installera som app

För bästa upplevelse, lägg till appen på hemskärmen:

  1. Tryck på dela-knappen i Safari
  2. Välj "Lägg till på hemskärmen"
  3. Tryck "Lägg till"
